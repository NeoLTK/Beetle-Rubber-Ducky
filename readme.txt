Beetle Rubber Ducky 🦆

🚀 Présentation
Beetle Rubber Ducky permet de transformer votre Beetle CJMCU (ou tout autre Atmega32u4) en un dispositif de type Rubber Ducky pour l'automatisation de tâches via des scripts. Ce projet offre une manière simple d'exécuter des scripts HID sur des ordinateurs en utilisant un microcontrôleur sans carte SD.

⚙️ Fonctionnalités
Compatible avec Atmega32u4 : Utilise le Beetle CJMCU ou d'autres microcontrôleurs Atmega32u4.

Sélection de script via broches : Permet de charger des scripts différents en fonction de la configuration des broches.

Optimisation : Code réécrit sous forme d'objets pour une utilisation plus facile et modulaire.

Aucune carte SD nécessaire : Facilité d'utilisation pour des projets simples et légers.

📦 Matériel requis
Beetle CJMCU (ou autre microcontrôleur Atmega32u4)

Câbles et outils pour programmer le microcontrôleur

📜 Installation
Clonez ce dépôt :
git clone https://github.com/NeoLTK/Beetle-Rubber-Ducky.git

Ouvrez le projet dans votre IDE (ex : Arduino IDE).

Connectez votre Beetle CJMCU à l'ordinateur.

Compilez et téléchargez le code sur le microcontrôleur.

Configurez les broches pour sélectionner les scripts.
